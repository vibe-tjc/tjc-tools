# 股票分析工具

根據股票名稱或代號，提供完整的股票研究報告。

## 功能

1. **公司基本資料** — 業務介紹、產業類別、市值、員工數等
2. **近期相關新聞** — 最近的重要新聞與市場動態
3. **最近兩季財報** — 營收、淨利、EPS、毛利率等關鍵指標

## 使用方式

告訴我你要分析的股票，例如：
- 「分析 TSMC 的股票」
- 「查看 AAPL 的財報」
- 「研究台積電的近期狀況」
- 「分析股票代號 2330」

## 研究流程

### 1. 查詢公司基本資料

```bash
./scripts/company-profile.sh "AAPL"
```

取得公司業務描述、產業分類、主要產品線、市值等基本資訊。

### 2. 搜尋近期新聞

```bash
./scripts/stock-news.sh "AAPL"
```

抓取該股票最近的相關新聞報導。

### 3. 查詢財報數據

```bash
./scripts/financials.sh "AAPL"
```

取得最近兩個季度的損益表關鍵數據。

### 4. 查詢即時報價（選用）

```bash
./scripts/quote.sh "AAPL"
```

取得目前股價、漲跌幅、成交量等即時資訊。

## 台灣股票支援

台灣上市櫃股票請在代號後加上 `.TW` 或 `.TWO`：
- 上市股票：`2330.TW`（台積電）、`2317.TW`（鴻海）
- 上櫃股票：`6510.TWO`（精測）

常見台股對照：

| 公司 | 代號 |
|------|------|
| 台積電 | 2330.TW |
| 鴻海 | 2317.TW |
| 聯發科 | 2454.TW |
| 台達電 | 2308.TW |
| 中華電 | 2412.TW |
| 大立光 | 3008.TW |

如果使用者只給了中文名稱或數字代號，請自動判斷並加上適當的後綴。

## 注意事項

- 資料來源以 Yahoo Finance 為主
- 財報數據可能有延遲，請以公司官方公告為準
- 本工具僅提供資訊整理，不構成投資建議
- 台灣股票的財報數據以新台幣為單位
- 美股財報數據以美元為單位
