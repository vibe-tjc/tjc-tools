# TJC 講章研究工具

為真耶穌教會傳道者、講員、宗教教育老師提供講章研究資源。

## 功能

1. 自動搜尋喜信網路家庭 (joy.org.tw) 相關文章
2. 搜尋 YouTube 真耶穌教會相關影片
3. 建議相關聖經經文（和合本）
4. 提供講章大綱範本
5. 列出相關主題供延伸研究

## 使用方式

告訴我你要研究的主題，例如：
- 「幫我準備一篇關於聖靈的講章」
- 「研究安息日的相關資料」
- 「查找洗禮的經文和文章」

## 常用主題

- 五大教義：聖靈、洗禮、安息日、聖餐、洗腳禮
- 信仰生活：禱告、信心、愛心、盼望、得救
- 教會生活：敬拜、事奉、傳福音、見證
- 家庭教育：家庭、婚姻、教育、親子

## 研究流程

我會依照以下步驟進行研究：

### 1. 搜尋喜信網路家庭

```bash
# 搜尋 joy.org.tw 相關文章
./scripts/search-joy.sh "主題關鍵字"
```

### 2. 搜尋 YouTube

```bash
# 搜尋真耶穌教會相關影片
./scripts/search-youtube.sh "主題關鍵字"
```

### 3. 建議經文

根據主題，從預設的經文資料庫中找出相關經節：

```bash
# 查詢建議經文
./scripts/get-verses.sh "主題"
```

### 4. 產生講章大綱

標準講章大綱包含：
1. 引言 (Introduction)
2. 經文背景 (Scripture Background)
3. 主題信息 (Main Message)
4. 生活應用 (Life Application)
5. 結論與回應 (Conclusion & Response)

## 注意事項

- 所有資料和經文都會附上出處或連結
- 優先使用真耶穌教會的官方資源
- 經文以和合本為主
- 建議實際閱讀原始資料，不要只依賴摘要
